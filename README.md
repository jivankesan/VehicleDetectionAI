# VehicleDetectionAI

## Overview

Created for the Toyota Innovation Hackathon, this project is an object detection program that identifies moving cars and their front wheels using OpenCV and image manipulation techniques.

## Features

- **Bounding Box**: Draws a bounding box around the moving car and its front wheel.
- **Image Manipulation**: Utilizes image manipulation techniques for accurate detection.

## Technologies

- OpenCV
- Python
